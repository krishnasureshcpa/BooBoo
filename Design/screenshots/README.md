# Screenshots

Place screenshots here.

## Naming Convention

```
app-screen-[view]-[version].png
```

### Examples

- `app-screen-main-1.0.png` — main window view, version 1.0
- `app-screen-menu-bar-1.0.png` — menu bar dropdown, version 1.0
- `app-screen-preferences-1.0.png` — preferences window, version 1.0
- `app-screen-notification-1.0.png` — notification example, version 1.0
- `app-screen-rule-detail-1.0.png` — individual rule detail view, version 1.0

## Versioning

Update screenshots when the UI changes. Keep previous version screenshots in the same directory for comparison.
