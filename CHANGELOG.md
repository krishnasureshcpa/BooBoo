# Changelog

All notable changes to BooBoo are documented here.

## [1.0.0] - 2026-07-01

### Added

- 50+ CIS Level 1 macOS security rules across 10 categories
- 5 probe types: password, system, sharing, file_system, launch
- Native SwiftUI GUI with dashboard, compliance list, and settings
- Terminal CLI (`booboo scan`, `booboo fix`, `booboo status`)
- Guided remediation with per-action confirmation
- Animated onboarding and built-in assistant
- Real-time scan progress and results
- Search and filter for compliance rules
- Keyboard shortcuts for all major actions
- Dark mode throughout
- MIT license

### Security

- Zero network connections by design
- No telemetry, analytics, or crash reporting
- All system changes require explicit user confirmation
